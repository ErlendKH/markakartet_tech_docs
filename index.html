    Markakartet (v2)

    Markakartet is an interactive map platform for managing and visualizing custom map layers.
    This second version builds on a first implementation made with JavaScript, HTML, and CSS, which worked well but had limitations when it came to scaling and maintainability.

    The backend is powered by Sanity, providing a flexible content platform for storing and editing map data.
    The map itself is built with OpenLayers, enabling advanced geospatial visualization and interaction.

    Why React?

    For this version, the project uses React in an attempt to take advantage of modern component-based architecture.
    React provides benefits such as:

    • Reusable UI components
    • Declarative state management
    • Potentially easier scaling for complex interfaces
